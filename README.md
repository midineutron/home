# Imminent Domain

This is a landing page and visualizer for the Imminent Domain project.
Code Exists on the `main` branch.
Bundled page exists on the `pages` branch.

## Tech Stack
- Three.js
- Tone.js
- vite
- Typescript

## Deployment Process
1. `npm run build`
2. Commit and push branch.
3. `npx push-dir --dir=dist --branch=pages`
